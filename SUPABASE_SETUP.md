# 「単語帳Web」Supabase 初期設定ガイド

本アプリケーションでインターネット同期および複数端末アクセスを行うには、バックエンドデータベース（クラウドDB）である **Supabase** の設定が必要です。
以下の手順に従って、無料枠で設定を完了させてください。

---

## 1. Supabaseプロジェクトの作成

1. [Supabase公式サイト](https://supabase.com/) にアクセスし、サインアップ（またはログイン）します。
2. ダッシュボードから **「New Project」** ボタンをクリックします。
3. 以下の項目を入力してプロジェクトを作成します：
   - **Name**: `TangoWeb`（任意の名前）
   - **Database Password**: 任意の安全なパスワードを入力（必ずメモしてください）
   - **Region**: 日本国内からのアクセスであれば、日本に一番近い **`Northeast Asia (Tokyo)`** を選択します。
   - **Pricing Plan**: **`Free`**（無料枠）を選択します。
4. 「Create new project」をクリックします。数分でプロジェクトの構築が完了します。

---

## 2. データベーステーブルとセキュリティポリシーの作成

プロジェクトが作成できたら、単語帳とカード用のテーブルを作成し、他のユーザーにデータが見られないようにするためのセキュリティ（RLS）設定を行います。

1. Supabaseの左メニューから **「SQL Editor」**（黒いスクエア内に `SQL` と書かれたアイコン）を選択します。
2. **「New query」**（または `New Query` プラスボタン）をクリックします。
3. プロジェクトのルートディレクトリにある **[supabase_schema.sql](file:///c:/Users/higen/Desktop/Coding/tango/supabase_schema.sql)** の中身をすべてコピーし、SQLエディタに貼り付けます。
4. エディタの右上にある **「Run」** ボタンをクリックして実行します。
   - 画面下部に `Success. No rows returned.` と表示されれば、テーブルおよびセキュリティポリシーの作成は成功です。

---

## 3. 環境変数の設定

次に、アプリケーションが作成したSupabaseプロジェクトに接続するためのキーを設定します。

1. 本プロジェクトのフォルダ内にある `.env.local.example` をコピーして、新しく **`.env.local`** ファイルを作成します。
2. Supabaseダッシュボードの左メニュー一番下にある **「Project Settings」**（歯車アイコン） -> **「API」** を開きます。
3. 画面に表示されている **Project API keys** と **Project URL** を確認します。
4. `.env.local` をテキストエディタで開き、以下のようにそれぞれの値を書き換えて保存します：

   ```env
   # Supabaseの接続設定
   NEXT_PUBLIC_SUPABASE_URL=【Project URL の値（https://...）】
   NEXT_PUBLIC_SUPABASE_ANON_KEY=【Project API keys の「anon public」の値】
   ```

---

## 4. 開発サーバーの起動

設定がすべて完了したら、アプリケーションを起動できます。

1. ターミナル（コマンドプロンプトやPowerShell）で以下のコマンドを実行し、開発用ローカルサーバーを起動します：
   ```bash
   npm run dev
   ```
2. ブラウザで `http://localhost:3000` にアクセスします。
3. ログイン画面が表示されます。
   - 「アカウント作成はこちら」をクリックし、テスト用のメールアドレスとパスワードを入力して新規登録を行ってください。
   - 登録が完了すると、自動的にダッシュボード画面へリダイレクトされます（※無料アカウントでは認証メールの確認をスキップして直接ログインできる設定になっています。もし登録後にログインできない場合は、入力した情報で「ログイン」をお試しください）。

---

## 5. 複数端末（スマホやiPad）でのアクセス方法

1. PCと同じWi-Fi（ローカルネットワーク）にスマートフォンやiPadを接続します。
2. PC側で `npm run dev` を起動しておきます。
3. PCのローカルIPアドレス（例: `192.168.x.x`）を調べます。
   - Windowsの場合: コマンドプロンプトで `ipconfig` を実行し、`IPv4 アドレス` の値を確認します。
4. スマホやiPadのブラウザを開き、`http://【PCのローカルIP】:3000`（例: `http://192.168.1.15:3000`）にアクセスします。
5. PC側で作成したアカウントと同じメールアドレス・パスワードでログインします。
6. インターネット上のデータベース（Supabase）を通じて、どちらの端末からカードを追加・更新しても、リアルタイムでデータが完全に同期されます。
